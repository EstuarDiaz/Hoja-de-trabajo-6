El map tiene una complejidad constante, ya que si el hash es bueno, no tendra colisiones y se hallara la posicion buscada en una sola operacion. Si el map llegara a repetirse, el map lo soluciona de tal forma que las operaciones para lidiar con las colisiones se mantene como un numero pequeño, por lo que lo podemos considerar constante. Esto varia segun la imlementacion, pero se obtienen resultados similares para cada una de ellas.
